Command: rm
Remove (unlink) the FILE(s) and directories.

Usage: rm [OPTION]... [FILE]...

[OPTION]
-f                      forcely remove everything
-i                      prompt before every removal
-I                      prompt before removing more than three files
--one-file-system       when removing a hierarchy recursively of same file system
--no-preserve-root      do not treat '/' (root directory) specially
-r, -R                  remove directories and their contents recursively
-d                      remove empty directories
-v                      remove file and directory and print message
