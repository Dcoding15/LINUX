Command: ncal
Displays a calendar and the date of Easter

Usage: ncal -C [general options] [-jy] [[month] year]
       ncal -C [general options] [-j] [-m month] [year]
       ncal [general options] [-bhJjpwySM] [-H yyyy-mm-dd] [-s country_code] [-W number of days] [[month] year]
       ncal [general options] [-Jeo] [year]

General options: [-31] [-A months] [-B months] [-d yyyy-mm]

[OPTION]

    -h                 Turns off highlighting of today.
    -m [no. of month]  Display the specified month.
    -p                 Print the country codes and switching days from Julian to Gregorian Calendar.
    -w                 Print the number of the week below each week column.
    -y yyyy            Display a calendar for the specified year. By default current year will displayed.
    -3                 Display previous month, current month, next month.
    -1                 Display only the current month. (By default)
    -A number          Display next 12 months + 1 current month.
    -B number          Display previous 12 months + 1 current month
    -d yyyy-mm         Use yyyy-mm as the current date (for debugging of date selection).
    -H yyyy-mm-dd      Use yyyy-mm-dd as the current date (for debugging of highlighting).
    -M                 Weeks start on Monday.
    -S                 Weeks start on Sunday. (By default)
    -W number          First week of the year has at least number days.
    -b                 Use oldstyle format with highlighting.
